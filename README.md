# Ebac-jdbc

Projeto desenvolvido para estudos de integração entre Java e MySQL utilizando JDBC.

O objetivo deste repositório é praticar operações de banco de dados diretamente em Java, aplicando conceitos fundamentais de backend como conexão com banco, CRUD, consultas SQL e boas práticas de segurança utilizando `PreparedStatement`.

## Tecnologias utilizadas

- Java 21
- MySQL
- JDBC
- MySQL Connector/J
- IntelliJ IDEA
- Git e GitHub

## Objetivo de aprendizado

Este projeto faz parte dos meus estudos em backend Java, com foco em desenvolvimento profissional utilizando banco de dados relacionais e boas práticas de programação.
